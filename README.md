# 长辈用药台账

一款面向家庭用药管理的原生微信小程序，支持长辈档案、用药记录、服药提醒和药物禁忌管理。

项目可离线运行，数据保存在当前设备的微信本地存储中，无需服务器、数据库、npm 依赖、第三方组件或云服务。

## 主要功能

- 概览与多家属切换
- 长辈档案新增、编辑、删除和详情查看
- 家属与长辈关联管理
- 药物主数据模糊匹配与维护
- 用药录入、记录编辑和删除
- 根据用药频次自动生成提醒规则
- 提醒状态筛选、已服和跳过操作
- 药物禁忌组合计算与风险筛选
- 适老化大字、高对比和振动提醒
- 数据字典展示
- 本地 JSON 备份与演示数据重置

## 运行方式

1. 打开微信开发者工具。
2. 选择“导入项目”，项目目录指向本仓库根目录。
3. `project.config.json` 已配置团队共用的小程序 AppID；协作成员需先由管理员授予开发者权限。使用其他小程序账号时，请替换为对应 AppID。
4. 点击“编译”即可运行。

## 数据说明

- `utils/seed.js`：内置长辈、家属、药物、用药记录、提醒规则、禁忌关系和数据字典等演示数据。
- `utils/database.js`：负责数据初始化、持久化、CRUD、关联查询、提醒生成和禁忌计算。
- `utils/api.js`：页面统一数据访问层。
- 数据通过 `wx.storage` 保存在当前设备。

可在“概览 → 更多管理 → 本地数据”中：

- 复制当前台账的 JSON 备份；
- 清除当前修改并恢复初始演示数据。

> 本地数据可能因卸载小程序、清理微信缓存或系统存储回收而丢失。当前存储方案适合单设备家庭使用，不支持多设备同步。

## 提醒方式

项目使用振动提示、大字提醒弹窗和高对比页面提供服药提醒，不使用在线语音服务。
